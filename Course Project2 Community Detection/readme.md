## README

- The clustering process is implemented by Louvain algorithm, which is stored in $\texttt{main.py}$ in folder $\texttt{src}$ .
- To execute the program, use the command $\texttt{python main.py}$.
- The file containing the edges must be put in $\texttt{edges.csv}$ in folder $\texttt{data}$.
- The file containing the ground truth must be put in $\texttt{ground_truth.csv}$ in folder $\texttt{data}$
- The result is put in $\texttt{answer.csv}$ in folder $\texttt{data}$.
